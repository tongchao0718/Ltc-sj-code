# YXUI 符合性审查清单

审查 marketing-demo 或营销域页面时逐项勾选。

## 布局与间距

- [ ] 顶栏 64px、页签栏 36px、侧栏 235px（收缩 54px 可交互）
- [ ] 侧栏/内容距页签 24px（面板整体下移）
- [ ] 侧栏↔内容 gap 24px（非 48px）
- [ ] 查询卡片↔列表卡片 gap **16px**
- [ ] 页面左/右/下外缘 24px
- [ ] 卡片内边距：上 20px，左右下 24px

## 查询区（#s14）

- [ ] 使用 `MdFilterForm`，非自写布局
- [ ] 三列栅格，标签左控件右
- [ ] 重置/查询在 Grid 内正确换行
- [ ] 次按钮 `#C6E2FF` 边框 + `#3A96FE` 字
- [ ] 控件高 28px、横间距 20px、按钮间距 10px（严格模式）

## 色彩与字体

- [ ] 主色 `#3A96FE` 及 hover/active
- [ ] 正文 14px、标题 16px、辅助 12px
- [ ] 列表表头 `#EDF5FF`
- [ ] 侧栏 inactive `#A3AFBC`、active `#3A96FE`

## 组件

- [ ] 分页 22×22，10/20/50，与查询联动
- [ ] 表格 loading/empty/error
- [ ] 弹窗：内容区居中、蒙版 50%、标题 48px、底栏右对齐
- [ ] 详情列表：标签右对齐、行高 36px
- [ ] 消息 Toast 四色背景
- [ ] 滚动条 6px 细窄

## 交互（§十）

- [ ] 查询变更重置页码并真实请求
- [ ] CRUD：确认 → loading → 结果 → 失败可恢复
- [ ] 防重复提交

## 壳层缺口（常见未通过）

- [ ] 侧栏收放 54px 可交互 + title Tooltip
- [ ] 蓝/绿换肤
- [ ] 面包屑（MdBreadcrumb）
- [ ] 弹窗内容区居中（#md-modal-host）
- [ ] 非列表页走标准双卡片结构
