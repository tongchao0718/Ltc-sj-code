# AI 智能 CRM 应用操作说明书

## 启动

```bash
npm run dev:ai-smart-crm-admin
```

入口：`/ai-smart-crm-admin/login` 或主应用应用中心。

## 构建

```bash
npm run build:ai-smart-crm-admin
```

## 功能模块

- 仪表盘、客户、线索、商机
- 数据导入、经营报表
- 销售阶段 / 公海配置、用户管理
- **APP 预览**：`/mobile/preview`（手机壳 Demo，非应用商店包）
