# 售电公司营销分析 Demo 应用操作说明书

## 前置条件

- Node.js >= 18，npm >= 9
- 仓库根目录执行：`npm install`

## 在主应用中使用

1. 启动主应用：`npm run dev`
2. 打开 **应用中心** → **营销系统 Demo**，或访问 `/marketing-demo/dashboard`
3. 左侧菜单按模块导航：数据接入、九维查询、多维分析、画像报告、结算协议、履约管控等
4. 接入校验：`npm run validate:sub-app-registry -- --app marketing-demo`

## 独立运行（开发）

```bash
npm run dev:marketing-demo
```

默认端口见终端输出（通常 **5177**），入口为 `/marketing-demo/dashboard`。

## 独立运行（构建后）

```bash
npm run build:marketing-demo
cd AIEP-WEB && npm run preview
```

产物目录：`AIEP-WEB/dist/marketing-demo/`。

## 核心操作说明

### 1. 综合查询（九维）

- 进入任一查询模块，使用顶部 **MdQueryLayout** 筛选区
- 支持重置、查询；列表支持分页与空态/错误态演示

### 2. 多维分析看板

- 从侧栏进入分析看板，点击图表可钻取至明细（Mock 链路）

### 3. 结算协议与履约

- 协议四环节：模板 → 生命周期 → 上传 → 审核（Mock 状态机）
- 履约五节点：临期提醒 → 流程处理 → 评估（Mock）

## 数据说明

- 全站 **Mock API**，契约见 `SDD-v1.0.json`
- 无生产环境数据，可安全演示

## 常见问题

| 现象 | 处理 |
|------|------|
| 白屏 | 检查 `npm install` 与控制台报错 |
| 404 | 确认 URL 含 `/marketing-demo` 前缀 |
| 样式异常 | 确认 YXUI 主题 CSS 已加载；可切换蓝/绿主题 |
