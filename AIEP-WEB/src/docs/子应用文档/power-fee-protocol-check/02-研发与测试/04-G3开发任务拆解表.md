- [ ] 代码提交已关联 `sdd_id` 与任务ID。

- [ ] 数据落库与数据库设计文档一致（字段、类型、主键、标准编码口径）。
- [ ] 模块状态机与异常恢复动作已实现并可演示。
- [ ] 单元测试/接口测试通过，AT 证据可追溯。
- [ ] 文档更新仅在“人工明确提出更新要求”后执行，并完成回写记录。
