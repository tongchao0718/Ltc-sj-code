# power-fee-protocol-check 应用操作说明书

## 1. 功能入口

与《02-产品设计文档》一级/二级菜单一致（侧栏进入）：

- 协议核查管理：协议模板库管理、协议样本标注管理、核查规则管理、核查任务策略管理  
- 异常分析及治理：协议识别核查执行、协议信息提取引擎、协议信息解析、识别结果校核、识别问题结果记录、问题分析及治理、核查任务监控、核查结果监控  
- 识别能力优化：协议识别能力优化、协同复核  
- 辅助验证：全链路验证（研发/联调用）

## 2. 操作流程

1. 先配置模板与规则；  
2. 再配置任务策略并发起核查；  
3. 查看异常并生成工单；  
4. 通过优化模块闭环改进。  

## 3. 常见问题

| 问题 | 处理建议 |
|------|------|
| 任务失败率高 | 检查规则版本与样本质量 |
| OCR错误率高 | 启动人机复核并补充样本 |
| 异常长期未闭环 | 检查工单分配与升级机制 |
