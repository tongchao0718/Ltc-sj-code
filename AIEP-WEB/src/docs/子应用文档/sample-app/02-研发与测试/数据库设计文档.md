## 2. 库表概览

### 2.1 功能域与表清单

| 功能域 | 模块 | 表名 | 表中文名 | 说明 |
|--------|------|------|----------|------|
| 工作台 | M01 | `dashboard_metric_snapshot` | 仪表盘指标快照 | 指标卡/图表数据源 |
| 列表查询 | M02 | `sample_list_item` | 列表示例数据 | 筛选、分页、排序 |
| 表单提交 | M03 | `sample_form_record` | 表单提交记录 | 幂等提交与状态 |
| 结果追踪 | M04 | `result_log` | 结果与异常日志 | 成功/失败/403/404/500 链路 |

### 2.2 表关系

```text
（无外键强依赖；四表独立，通过 request_id / trace_id 做链路关联）
sample_form_record.request_id ──► result_log.request_id
```

### 2.3 公共字段约定

除 `result_log` 外，业务表均含 `created_at`、`updated_at`；落地模式增加 `created_by`、`updated_by`、`is_deleted`、`version`。

## 8. SQL 清单（全量 DDL）

```sql
-- 方言：MySQL 8.x · Schema: aiep_sample

CREATE TABLE dashboard_metric_snapshot (
  dashboard_metric_snapshot_id BIGINT NOT NULL AUTO_INCREMENT COMMENT '快照主键ID',
  metric_code                  VARCHAR(64)    NOT NULL COMMENT '指标编码',
  metric_name                  VARCHAR(128)   NOT NULL COMMENT '指标名称',
  metric_value                 DECIMAL(18,4)  NOT NULL DEFAULT 0 COMMENT '指标值',
  unit                         VARCHAR(32)    NULL COMMENT '单位',
  snapshot_date                DATE           NOT NULL COMMENT '快照日期',
  request_id                   VARCHAR(64)    NULL COMMENT '请求追踪号',
  created_at                   DATETIME       NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  PRIMARY KEY (dashboard_metric_snapshot_id),
  UNIQUE KEY uk_dms_metric_date (metric_code, snapshot_date),
  KEY idx_dms_snapshot_date (snapshot_date)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='仪表盘指标快照';

CREATE TABLE sample_list_item (
  sample_list_item_id BIGINT NOT NULL AUTO_INCREMENT COMMENT '列表项主键ID',
  item_code           VARCHAR(64)  NOT NULL COMMENT '项目编码',
  item_name           VARCHAR(128) NOT NULL COMMENT '项目名称',
  category            VARCHAR(32)  NULL COMMENT '分类',
  status              VARCHAR(20)  NOT NULL DEFAULT 'active' COMMENT '状态 LIST_ITEM_STATUS',
  sort_order          INT          NOT NULL DEFAULT 0 COMMENT '排序号',
  created_at          DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  updated_at          DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  PRIMARY KEY (sample_list_item_id),
  UNIQUE KEY uk_sli_item_code (item_code),
  KEY idx_sli_status_updated (status, updated_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='列表示例数据';

CREATE TABLE sample_form_record (
  sample_form_record_id BIGINT NOT NULL AUTO_INCREMENT COMMENT '提交记录主键ID',
  submit_id             VARCHAR(64)  NOT NULL COMMENT '幂等提交号',
  form_type             VARCHAR(32)  NOT NULL DEFAULT 'step_form' COMMENT '表单类型',
  form_payload          JSON         NOT NULL COMMENT '表单内容',
  status                VARCHAR(20)  NOT NULL DEFAULT 'pending' COMMENT '提交状态 FORM_SUBMIT_STATUS',
  request_id            VARCHAR(64)  NOT NULL COMMENT '请求追踪号',
  error_message         VARCHAR(512) NULL COMMENT '失败原因',
  created_at            DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  updated_at            DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  PRIMARY KEY (sample_form_record_id),
  UNIQUE KEY uk_sfr_submit_id (submit_id),
  KEY idx_sfr_status_created (status, created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='表单提交记录';

CREATE TABLE result_log (
  result_log_id  BIGINT NOT NULL AUTO_INCREMENT COMMENT '日志主键ID',
  request_id     VARCHAR(64)  NOT NULL COMMENT '请求追踪号',
  trace_id       VARCHAR(64)  NULL COMMENT '链路追踪号',
  page_route     VARCHAR(128) NOT NULL COMMENT '页面路由',
  result_type    VARCHAR(32)  NOT NULL COMMENT '结果类型 RESULT_TYPE',
  http_status    SMALLINT     NULL COMMENT 'HTTP状态码',
  error_code     VARCHAR(32)  NULL COMMENT '业务错误码',
  error_message  TEXT         NULL COMMENT '错误信息',
  created_at     DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  PRIMARY KEY (result_log_id),
  UNIQUE KEY uk_rl_request_id (request_id),
  KEY idx_rl_result_created (result_type, created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='结果与异常日志';
```
