# sample-app 数据库设计文档

## 1. 表清单

| 表名 | 用途 | 关联模块 |
|------|------|------|
| dashboard_metric_snapshot | 仪表盘指标快照 | M01 |
| sample_list_item | 列表查询数据 | M02 |
| sample_form_record | 表单提交记录 | M03 |
| result_log | 结果与异常日志 | M04 |

## 2. 核心字段（示例）

| 表名 | 字段 | 类型 | 说明 |
|------|------|------|------|
| sample_list_item | id | bigint | 主键 |
| sample_list_item | status | varchar(32) | 状态编码 |
| sample_form_record | submit_id | varchar(64) | 幂等提交号 |
| result_log | request_id | varchar(64) | 请求追踪号 |

## 3. 约束规则

- 所有业务主键不可复用。
- 关键字段（status/type）必须使用统一编码。
- 日志表保留 request_id 以支持全链路追踪。
