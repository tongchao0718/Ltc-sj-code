# sample-app 应用操作说明书

## 1. 启动与访问

- 开发启动：`npm run dev:sample-app`
- 构建命令：`npm run build:sample-app`
- 访问入口：`/sample-app`

## 2. 常用操作

1. 进入工作台查看核心指标。  
2. 在查询页使用关键词、状态筛选并翻页。  
3. 在表单页提交数据并查看结果反馈。  

## 3. 常见问题

| 问题 | 现象 | 处理建议 |
|------|------|------|
| 列表无数据 | 页面为空 | 检查筛选条件与接口返回 |
| 提交失败 | 出现错误提示 | 检查字段校验与网络状态 |
| 页面异常跳转 | 进入 404/500 | 按返回入口回退并查看日志 |

## 4. 发布检查项

- 构建产物可访问；
- 核心路由可访问；
- G2/G3 门禁记录完整；
- 验收脚本执行通过。
