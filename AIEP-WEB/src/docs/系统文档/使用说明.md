# 系统使用说明

## 1. 安装

```bash
npm install
```

## 2. 开发

```bash
npm run dev
```

默认访问 `http://localhost:5173`。

## 3. 示例子应用独立调试

```bash
npm run dev:sample-app
```

按终端提示打开子应用入口页面（通常为 `/build/sample-app.html`）。

## 4. 构建

- 主应用：`npm run build` → `dist/`
- 子应用：`npm run build:sample-app` → `dist/sample-app/`

## 5. 预览

```bash
npm run preview
```

## 6. 部署

主应用：部署 `dist/` 并配置 SPA 回退。子应用：部署 `dist/sample-app/` 或随主应用一体部署。

## 7. 系统功能

### 主应用功能
- **首页**：系统概览
- **应用中心**：管理和访问子应用
- **关于页面**：系统信息
- **帮助页面**：使用帮助
- **个人中心**：用户信息管理
- **审核中心**：内容审核

### 示例应用功能
- **仪表盘**：工作台、实时监控
- **数据可视化**：数据分析、多维数据分析
- **列表页**：查询表格、卡片列表
- **表单页**：分步表单、分组表单
- **详情页**：基础详情页
- **结果页**：成功页、失败页
- **异常页**：403、404、500 页面
- **个人中心**：个人资料、用户设置

## 8. 操作流程

### 访问示例应用
1. 启动开发服务器：`npm run dev`
2. 打开浏览器，访问 `http://localhost:5173`
3. 点击导航栏中的「应用中心」
4. 在应用中心页面，找到「示例应用」卡片，点击「进入应用」按钮
5. 进入示例应用后，使用左侧菜单切换不同功能模块

### 使用示例应用功能
1. **工作台**：查看数据统计、内容数据图表、线上热门内容和内容类型占比
2. **数据可视化**：查看数据分析和多维数据分析页面的图表
3. **列表页**：使用查询表格和卡片列表功能
4. **表单页**：填写分步表单和分组表单
5. **个人中心**：查看和修改个人资料、用户设置

## 9. 常见问题

### 端口占用
如果端口 5173 被占用，开发服务器会自动使用其他端口（如 5174、5175 等）。

### 子应用加载失败
如果子应用加载失败，请检查：
1. 子应用的依赖是否安装
2. 子应用的代码是否有语法错误
3. 主应用的路由配置是否正确

### 构建失败
如果构建失败，请检查：
1. 代码是否有语法错误
2. 依赖是否安装正确
3. 构建配置是否正确
