# 目录结构说明

```
Project-sj/
├── build/                    # 子应用独立构建：HTML 入口、Vite 配置、start 模板
├── backups/                  # 一键更新脚本备份占位
├── versions/                 # 版本备份占位（可按流程扩展）
├── docs/
│   ├── 系统文档/             # 本目录：系统级需求、设计、接口等
│   └── 子应用文档/           # 各子应用文档
├── public/                   # 静态资源（如 vite.svg）
├── scripts/                  # update.js、rollback.js（ESM）
├── src/
│   ├── apps/sample-app/      # 示例子应用（布局、页面、独立 main.js）
│   ├── components/           # 公共组件
│   ├── router/               # 主应用路由
│   ├── utils/                # 工具（如 errorHandler）
│   ├── views/                # 主应用页面
│   ├── App.vue
│   ├── main.js
│   └── style.css
├── dist/                     # 构建输出（勿提交，见 .gitignore）
├── index.html                # 主应用 HTML 入口
├── vite.config.js            # 主应用 Vite 配置
├── package.json
└── *.md                      # README、创建指南、打包指南等
```

## 构建产物

- 主应用：`dist/`（`index.html` + `assets/`）
- 示例子应用：`dist/sample-app/`（`index.html`、`sample-app.html`、`start.html`、`assets/`）
