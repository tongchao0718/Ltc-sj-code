# 系统详细设计文档

## 1. 路由设计（主应用）

- `/` → 重定向 `/dashboard`
- `/dashboard`：系统首页
- `/app-center`：应用中心
- `/sample-app/*`：示例子应用嵌套路由
  - `/sample-app/dashboard/workplace`：工作台
  - `/sample-app/dashboard/monitor`：实时监控
  - `/sample-app/visualization/data-analysis`：数据分析
  - `/sample-app/visualization/multi-dimension-data-analysis`：多维数据分析
  - `/sample-app/list/search-table`：查询表格
  - `/sample-app/list/card`：卡片列表
  - `/sample-app/form/step`：分步表单
  - `/sample-app/form/group`：分组表单
  - `/sample-app/profile/basic`：基础详情页
  - `/sample-app/result/success`：成功页
  - `/sample-app/result/error`：失败页
  - `/sample-app/exception/403`：403 页面
  - `/sample-app/exception/404`：404 页面
  - `/sample-app/exception/500`：500 页面
  - `/sample-app/user/info`：个人资料
  - `/sample-app/user/setting`：用户设置

路由定义见 `src/router/index.js`。

## 2. 路由设计（子应用独立包）

入口 `src/apps/sample-app/main.js`，`createWebHashHistory()`，路径为 `/dashboard/workplace`、`/visualization/data-analysis` 等，不含 `sample-app` 前缀。

## 3. 路径前缀策略

`SampleApp.vue` 根据当前 `route.path` 是否以 `/sample-app` 开头，动态拼接链接，保证**主应用内嵌**与**独立运行**一致。

## 4. 打包插件

`build/vite.sample-app.config.js` 中 `copyHtmlPlugin` 在构建结束后整理 `dist/sample-app` 下 HTML，并生成 `start.html`（CDN 备选）。

## 5. 错误处理

`src/utils/errorHandler.js` 提供统一 `reportError` 钩子，可扩展对接监控。

## 6. 应用结构

### 主应用结构
- `src/App.vue`：主应用根组件
- `src/main.js`：主应用入口
- `src/router/index.js`：主应用路由配置
- `src/views/`：主应用页面组件
  - `Dashboard.vue`：系统首页
  - `AppCenter.vue`：应用中心
  - `AboutPage.vue`：关于页面
  - `HelpPage.vue`：帮助页面
  - `ProfileCenter.vue`：个人中心
  - `ReviewCenter.vue`：审核中心
- `src/components/`：主应用通用组件
- `src/utils/`：主应用工具函数
- `src/style.css`：主应用全局样式

### 子应用结构
- `src/apps/sample-app/`：示例应用
  - `SampleApp.vue`：示例应用根组件
  - `main.js`：示例应用入口
  - `i18n.js`：示例应用国际化配置
  - `views/`：示例应用页面组件
  - `components/`：示例应用通用组件
  - `utils/`：示例应用工具函数
  - `styles/`：示例应用样式
  - `locales/`：示例应用国际化资源

## 7. 功能模块

### 仪表盘模块
- 工作台：数据统计、内容数据图表、线上热门内容、内容类型占比
- 实时监控：系统运行状态监控

### 数据可视化模块
- 数据分析：多种图表展示，包括折线图、柱状图等
- 多维数据分析：复杂数据可视化，支持多维度数据展示

### 列表页模块
- 查询表格：支持筛选、搜索、分页等功能
- 卡片列表：支持标签切换和卡片展示

### 表单页模块
- 分步表单：支持表单验证和提交
- 分组表单：支持表单分组和复杂表单逻辑

### 详情页模块
- 基础详情页：展示详细信息

### 结果页模块
- 成功页：操作成功提示
- 失败页：操作失败提示

### 异常页模块
- 403 页面：权限不足提示
- 404 页面：页面不存在提示
- 500 页面：服务器错误提示

### 个人中心模块
- 个人资料：用户个人信息管理
- 用户设置：用户账号设置

## 8. 样式设计

- 参考 Arco Design Pro 的设计风格
- 使用蓝色主色调（#165DFF）
- 左侧菜单背景为白色
- 内容区域背景为白色，卡片和表格具有适当的阴影和边框
- 响应式设计，适配不同屏幕尺寸

## 9. 国际化支持

- 主应用和子应用均支持国际化
- 示例应用包含多种语言的翻译文件
- 可通过 i18n 配置切换语言
