# PRD 页面标注 — 聚合示例

## 列表示例页

| UI 区域 | PRD 片段（示例） | 聚合策略 | 锚点位置 |
|---------|------------------|----------|----------|
| 筛选区 | 查询字段、默认值、联动、查询/重置 | 合并为 `[1]` | 筛选卡片右上角 |
| 表格列 | 列定义、排序、格式化 | 按「表格主体」合并为 `[2]` | 表头区域右上角 |
| 行操作 | 编辑、删除、查看、权限 | 合并为 `[3]` | 「操作」列标题旁 |
| 分页 | 页大小、跳转、总数展示 | 合并为 `[4]` | 分页条右上角 |
| 批量工具栏 | 导出、批量删除、审批 | 合并为 `[5]` | 工具栏容器右上角 |

**反例：** 在「编辑」「删除」按钮上各放一个角标 — 禁止。

## 详情 / 抽屉

| UI 区域 | 说明 |
|---------|------|
| 头部摘要区 | 标题、状态标签、关键 KPI 合并 |

## 浮窗内容组织示例（节选）

角标 `[3]` — 行操作，浮窗内应包含 PRD 原文全部条目，例如：

```markdown
**需求描述：列表行操作**

### 显示样式
- 操作列固定右侧，宽度不小于 160px
- 按钮：编辑（主色链接）、删除（危险色，需二次确认）

### 交互与排序
1. 无权限时隐藏对应按钮，不占位
2. 删除成功后刷新当前页，不跳转

### 业务定义
- **编辑**：跳转 `/detail/:id?mode=edit`
- **查看**：只读模式打开抽屉

### 备注
> 权限码与菜单配置表 4.2 节一致
```

注意：以上为结构示例；实际内容必须从 `prd.md` 零删减迁移。

## Workflow B diff 示例

| 变更类型 | 页面 | prd.md | 动作 |
|----------|------|--------|------|
| 修改 | 浮窗文案 | 段落改写 | 只更新 `markdown` 映射与 prd 正文，不动角标 CSS |
| 新增 | 新筛选字段 | 新增小节 | 新编号 `max+1`，新锚点，写回 `[n]` |
| 删除 | 移除导出按钮 | 删除需求 | 移除角标与映射，删除 prd 中 `[n]` |
