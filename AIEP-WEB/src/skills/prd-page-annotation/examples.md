# PRD 页面标注 — 示例

## 按功能点标注（同页多角标）

**页面**：`SearchTablePage`（一页 **两个** 功能点）

| 角标 id | featureId | 功能点 | 锚点位置 |
|---------|-----------|--------|----------|
| 2 | SA-2 | 列表筛选与查询 | 筛选区 `.filter` 右上角 |
| 3 | SA-2 | 列表表格与分页 | 表格区右上角 |

**反例**：在「查询」「重置」各放一个角标 — 应合并为功能点「筛选与查询」一个角标。

## 功能点 vs 整页

| 粒度 | 适用 | 示例 |
|------|------|------|
| 功能点（推荐） | 可独立验收、可独立走查 | 筛选、表格、行操作、侧栏导航 |
| 整页 | 单功能简单页 | 成功页仅「结果展示 + 返回」 |

## 四段式内容示例（功能点：列表筛选）

```javascript
sections: {
  functional:
    '筛选字段：编号、名称、状态；查询/重置按钮；与表格区分割线。',
  interaction:
    '1. 查询带筛选参数请求列表。\n2. 重置清空并查询。\n3. 筛选变更后分页归 1。',
  dataModel:
    '表 sample_list_item；GET /api/list/query（详见 SDD §3.2）；状态枚举见 PRD 第七章。',
  other: '验收 AT-02；NFR-3 错误须提示。'
}
```

## 列表页功能点拆分模板

| 功能点 | 典型锚点 |
|--------|----------|
| 筛选与查询 | 筛选卡片右上角 |
| 表格与列展示 | 表头区域 |
| 行操作 | 「操作」列标题 |
| 批量工具栏 | 工具栏容器 |
| 分页 | 分页条 |

每个功能点 **一个** 角标、**一套** 四段式。

## Workflow B diff

| 变更 | annotations | 页面 Vue | 文档 |
|------|-------------|----------|------|
| 修改交互 | 改 `sections.interaction` | 不动 | 同步 03-PRD |
| 新增功能点 | 新 id + sections | 新 `PrdAnnotationAnchor` | 01/03 加 SA |
| 删除功能点 | 删键 | 删锚点 | 删文档行 |

## 浮窗编辑流程（人工）

1. `npm run dev` 打开 sample-app  
2. Hover 角标 → 浮窗 → **编辑**  
3. 改四段式 → **保存到本地**（刷新仍生效）  
4. **复制代码片段** → 贴入 `sampleAppAnnotations.js` → 提交
