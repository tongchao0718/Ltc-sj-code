# 示例应用操作说明

## 在主应用中使用

1. 启动：`npm run dev`
2. 打开应用中心 →「进入应用」，或访问 `/sample-app`
3. 使用左侧菜单切换「首页」「协议管理」「功能模块」

## 独立运行（开发）

```bash
npm run dev:sample-app
```

浏览器访问开发服务器提示的地址，打开 `/build/sample-app.html`。

## 独立运行（构建后）

```bash
npm run build:sample-app
```

在资源管理器中打开 `dist/sample-app/index.html`（推荐使用 `npm run preview` 或静态服务器以避免浏览器本地策略限制）。
