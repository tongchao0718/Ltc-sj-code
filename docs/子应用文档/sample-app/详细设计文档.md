# 示例应用详细设计

## 1. 组件结构

- `SampleApp.vue`：侧栏 + `<router-view>`
- `views/Home.vue`、`Module*.vue`、`protocol/*`：页面级组件
- `main.js`：独立运行时的 Hash 路由表

## 2. 路由对照

| 场景 | 协议总览路径示例 |
|------|------------------|
| 主应用嵌套 | `/sample-app/protocol/overview` |
| 独立包 | `#/protocol/overview` |

## 3. 样式

依赖全局 `src/style.css` 中 CSS 变量；组件内 `scoped` 样式补充布局。

## 4. 构建

见 `build/vite.sample-app.config.js` 与《子应用打包指南》。
