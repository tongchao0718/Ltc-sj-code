# 示例应用需求说明书

## 1. 范围

示例子应用 `sample-app`，包含页面与路由见 `src/apps/sample-app/` 与主路由 `/sample-app`。

## 2. 功能需求

| 编号 | 需求 | 验收 |
|------|------|------|
| SA-1 | 首页展示欢迎语与三个模块入口 | 链接可跳转 |
| SA-2 | 协议管理五页可访问 | 侧栏与路由一致 |
| SA-3 | 模块 1～3 占位内容可访问 | 页面可渲染 |
| SA-4 | 独立打包可本地打开 | `dist/sample-app/index.html` |

## 3. 非功能

响应式侧栏在窄屏下纵向排列；样式与主应用 CSS 变量一致。
