## 独立运行（开发）

```bash
npm run dev:<app-code>
```

访问终端提示地址，并打开对应子应用入口页面。

## 独立运行（构建后）

```bash
npm run build:<app-code>
```

打开构建产物页面（建议使用 `npm run preview` 或静态服务器）。

## 常见问题

- 页面白屏：检查依赖安装与控制台资源报错。
- 路由无法访问：确认路径包含 `<主路由前缀>`。
- 本地文件访问限制：优先使用 `npm run preview`。
