# 系统概要设计文档

## 1. 架构概述

采用**单仓库、多入口**方式：

- **主应用**：`index.html` → `src/main.js`，Vue Router History 模式。
- **子应用**：源码位于 `src/apps/<name>/`，独立入口 `main.js`，独立构建使用 Hash 模式以便 `file://` 打开。

## 2. 模块划分

| 模块 | 路径 | 说明 |
|------|------|------|
| 壳层 | `src/App.vue`、`src/router` | 布局与顶层路由 |
| 主视图 | `src/views/` | Dashboard、应用中心 |
| 示例子应用 | `src/apps/sample-app/` | 侧边栏 + 业务页 |
| 公共组件 | `src/components/` | 帮助、关于等 |

## 3. 技术栈

Vue 3、Vue Router 4、Vite 6、原生 CSS 变量。

## 4. 部署视图

- 主站点：静态托管 `dist/`。
- 子应用独立站点：静态托管 `dist/sample-app/`。

## 5. 扩展点

新增子应用：增加目录、`build/*.html` 与 `vite.*.config.js`、注册路由与应用中心卡片。
