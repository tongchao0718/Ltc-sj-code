# 系统详细设计文档

## 1. 路由设计（主应用）

- `/` → 重定向 `/dashboard`
- `/dashboard`：系统首页
- `/app-center`：应用中心
- `/sample-app/*`：示例子应用嵌套路由（含 `protocol` 子树）

路由定义见 `src/router/index.js`。

## 2. 路由设计（子应用独立包）

入口 `src/apps/sample-app/main.js`，`createWebHashHistory()`，路径为 `/home`、`/protocol/...`、`/module1` 等，不含 `sample-app` 前缀。

## 3. 路径前缀策略

`SampleApp.vue` 与 `Home.vue` 根据当前 `route.path` 是否以 `/sample-app` 开头，动态拼接链接，保证**主应用内嵌**与**独立运行**一致。

## 4. 打包插件

`build/vite.sample-app.config.js` 中 `copyHtmlPlugin` 在构建结束后整理 `dist/sample-app` 下 HTML，并生成 `start.html`（CDN 备选）。

## 5. 错误处理

`src/utils/errorHandler.js` 提供统一 `reportError` 钩子，可扩展对接监控。
