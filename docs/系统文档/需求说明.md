# 系统需求说明

## 1. 概述

本系统为多子应用管理平台（Vue 3 + Vite），支持主应用统一入口与子应用独立打包部署。

## 2. 功能需求

- 主应用：Dashboard、应用中心、顶栏导航。
- 子应用：示例子应用 `sample-app`（首页、协议管理、功能模块等），可扩展。
- 构建：主应用输出 `dist/`；子应用输出 `dist/sample-app/`。

## 3. 非功能需求

- 浏览器兼容：现代浏览器；不支持 IE11。
- 可维护性：遵循 kebab-case 子应用目录与统一打包规范。

## 4. 约束

须符合《子应用打包指南》《主应用打包指南》中的验证与部署要求。
