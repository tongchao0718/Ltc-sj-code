# 示例应用详细设计

## 1. 组件结构

- `SampleApp.vue`：侧栏菜单 + `<router-view>`，包含菜单管理和路由导航逻辑
- `views/` 目录：包含所有页面级组件
  - `dashboard/`：仪表盘相关页面
    - `WorkplacePage.vue`：工作台页面，包含数据统计、图表展示等功能
  - `visualization/`：数据可视化相关页面
    - `DataAnalysisPage.vue`：数据分析页面，包含多种图表展示
    - `MultiDimensionPage.vue`：多维数据分析页面，包含复杂数据可视化
  - `list/`：列表相关页面
    - `SearchTablePage.vue`：查询表格页面，包含筛选、搜索、分页等功能
    - `CardListPage.vue`：卡片列表页面，包含标签切换和卡片展示
  - `form/`：表单相关页面
    - `StepFormPage.vue`：分步表单页面
  - `result/`：结果相关页面
    - `SuccessPage.vue`：成功页
    - `ErrorPage.vue`：失败页
  - `exception/`：异常相关页面
    - `ForbiddenPage.vue`：403 页面
    - `NotFoundPage.vue`：404 页面
    - `ServerErrorPage.vue`：500 页面
  - `user/`：用户相关页面
    - `UserInfo.vue`：个人资料页面
    - `UserSetting.vue`：用户设置页面
- `components/` 目录：包含可复用组件
  - `ArcoDemoShell.vue`：示例应用外壳组件
  - `ArcoBreadcrumb.vue`：面包屑组件
  - `EChart.vue`：图表组件，基于 ECharts
- `utils/` 目录：工具函数
  - `chartTheme.js`：图表主题配置
- `styles/` 目录：样式文件
  - `sample-arco.css`：示例应用样式
- `locales/` 目录：国际化资源
  - 包含多种语言的翻译文件
- `main.js`：应用入口，配置路由和应用初始化
- `i18n.js`：国际化配置

## 3. 样式

- 依赖全局 `src/style.css` 中 CSS 变量
- 组件内 `scoped` 样式补充布局
- 左侧菜单背景为白色，与 Arco Design Pro 一致
- 内容区域背景为白色，卡片和表格具有适当的阴影和边框

## 4. 功能特性

- **仪表盘**：展示数据统计、内容数据图表、线上热门内容和内容类型占比
- **数据可视化**：提供多种图表展示，包括折线图、柱状图、饼图等
- **列表页**：包含查询表格和卡片列表，支持筛选、搜索和分页
- **表单页**：提供分步表单，支持表单验证和提交
- **结果页**：展示操作成功和失败的结果
- **异常页**：展示 403、404 和 500 错误页面
- **用户中心**：提供个人资料和用户设置功能

## 6. 关键页面时序（最小实现）

### 6.1 查询表格页（`/sample-app/list/search-table`）

1. 页面初始化：加载默认筛选项和第一页数据。  
2. 用户操作：修改筛选项并触发查询。  
3. 请求发送：携带 `pageNo/pageSize` 和筛选参数请求列表接口。  
4. 结果处理：成功时刷新表格和分页；失败时提示错误并保留筛选条件。  
5. 后续动作：支持分页、排序、查看详情、导入导出等扩展操作。  

### 6.2 分步表单页（`/sample-app/form/step`）

1. 页面初始化：读取草稿（如有）并恢复步骤。  
2. 用户填写：前端实时校验字段完整性和格式。  
3. 步骤切换：当前步骤校验通过后允许进入下一步。  
4. 提交处理：提交成功跳转结果页；失败保留输入并显示错误原因。  
5. 幂等控制：同一步骤重复提交使用 `draftId + stepNo` 去重。  

## 7. 状态机与异常处理（最小实现）

### 7.1 通用页面状态

- `idle`：初始态，页面待交互。  
- `loading`：请求中，展示骨架屏或加载态。  
- `success`：请求成功并渲染业务数据。  
- `empty`：请求成功但无数据，展示空态。  
- `error`：请求失败，展示错误提示和重试入口。  

### 7.2 关键异常与处理策略

| 场景 | 触发条件 | 处理方式 | 用户可见反馈 |
|------|------|------|------|
| 参数校验失败 | 前端校验不通过或后端返回 `4001` | 中断提交，定位错误字段 | 表单字段级错误提示 |
| 权限不足 | 后端返回 `4030` 或路由无权限 | 跳转 403 页面并保留回退入口 | “无权限访问”提示 |
| 资源不存在 | 访问无效详情/路由 | 跳转 404 页面 | “页面不存在”提示 |
| 服务异常 | 后端返回 `5000` 或网络异常 | 展示错误态并允许重试 | “服务异常，请稍后重试” |
