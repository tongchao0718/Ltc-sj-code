# 依赖安装指南

## 1. 一键安装方案

### 1.1 核心依赖安装（带检测）

```bash
# 检测并安装核心依赖
if ! npm list vue@3 &> /dev/null || ! npm list vue-router@4 &> /dev/null; then
  echo "正在安装核心依赖..."
  npm install vue@3 vue-router@4
else
  echo "核心依赖已安装"
fi
```

### 1.2 开发依赖安装（带检测）

```bash
# 检测并安装开发依赖
if ! npm list --save-dev vite &> /dev/null || ! npm list --save-dev @vitejs/plugin-vue &> /dev/null || ! npm list --save-dev terser &> /dev/null; then
  echo "正在安装开发依赖..."
  npm install --save-dev vite @vitejs/plugin-vue terser
else
  echo "开发依赖已安装"
fi
```

### 1.3 可选依赖安装（根据需要，带检测）

```bash
# PWA 支持
if ! npm list --save-dev vite-plugin-pwa &> /dev/null; then
  echo "正在安装 PWA 支持..."
  npm install --save-dev vite-plugin-pwa
else
  echo "PWA 支持已安装"
fi

# 国际化支持
if ! npm list vue-i18n@next &> /dev/null; then
  echo "正在安装国际化支持..."
  npm install vue-i18n@next
else
  echo "国际化支持已安装"
fi

# 代码质量工具
if ! npm list --save-dev eslint &> /dev/null || ! npm list --save-dev eslint-plugin-vue &> /dev/null || ! npm list --save-dev prettier &> /dev/null; then
  echo "正在安装代码质量工具..."
  npm install --save-dev eslint eslint-plugin-vue prettier
else
  echo "代码质量工具已安装"
fi
```

### 1.4 一键安装所有依赖（带检测）

```bash
# 检测并安装核心依赖和开发依赖
if ! npm list vue@3 &> /dev/null || ! npm list vue-router@4 &> /dev/null || ! npm list --save-dev vite &> /dev/null || ! npm list --save-dev @vitejs/plugin-vue &> /dev/null || ! npm list --save-dev terser &> /dev/null; then
  echo "正在安装核心依赖和开发依赖..."
  npm install vue@3 vue-router@4 vite @vitejs/plugin-vue terser
else
  echo "核心依赖和开发依赖已安装"
fi

# 可选：检测并安装所有可选依赖
if ! npm list --save-dev vite-plugin-pwa &> /dev/null || ! npm list vue-i18n@next &> /dev/null || ! npm list --save-dev eslint &> /dev/null || ! npm list --save-dev eslint-plugin-vue &> /dev/null || ! npm list --save-dev prettier &> /dev/null; then
  echo "正在安装可选依赖..."
  npm install --save-dev vite-plugin-pwa && npm install vue-i18n@next && npm install --save-dev eslint eslint-plugin-vue prettier
else
  echo "可选依赖已安装"
fi
```

## 3. 安装步骤

1. **打开命令行终端**：在项目根目录下打开命令行终端
2. **执行安装命令**：根据需要执行上述安装命令
3. **验证安装**：安装完成后，运行 `npm list` 检查依赖是否正确安装
4. **启动项目**：安装完成后，可以运行 `npm run dev` 启动开发服务器

## 4. 注意事项

- 确保 Node.js 版本为 16.0 或更高版本
- 确保 npm 版本为 8.0 或更高版本
- 如果安装过程中遇到网络问题，可以尝试使用 npm 镜像源
- 安装完成后，建议运行 `npm run build` 测试构建是否正常

## 5. 故障排除

### 5.1 依赖冲突

如果遇到依赖冲突，可以尝试：

```bash
# 清除 npm 缓存
npm cache clean --force

# 删除 node_modules 目录和 package-lock.json 文件
rm -rf node_modules package-lock.json

# 重新安装依赖
npm install
```

### 5.2 安装失败

如果安装失败，可以尝试：

```bash
# 使用 npm 镜像源
npm install --registry=https://registry.npmmirror.com

# 或使用 yarn
npm install -g yarn
yarn install
```

## 6. 总结

通过本指南提供的一键安装命令，您可以快速安装项目所需的所有依赖，确保项目能够正常构建和运行。根据项目需求，您可以选择安装相应的可选依赖，以增强项目的功能和质量。
