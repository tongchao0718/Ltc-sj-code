# 应用系统文档清单

## 0. 仓库根目录开发工具配置（不可合并）

以下目录位于 **仓库根目录**，由 GitHub / VS Code / Cursor 各自约定路径读取，**禁止**合并为单一目录。

| 目录 | 必要性 | 作用 | 维护说明 |
|------|--------|------|----------|
| `.github/` | 必要（使用 GitHub 时） | CI（`workflows/sdd-gate.yml`）、PR 模板 | 仅 GitHub 平台识别此路径 |
| `.vscode/` | 建议保留 | 编辑器推荐扩展（`extensions.json`） | Cursor 继承 VS Code 工作区配置 |
| `.cursor/` | 必要 | Agent Skills（`ai-full-process-design` 等）、Rules（`.mdc`，含 **agent-skills-auto-exec 自动加载**） | AI+ 全流程默认入口；**postinstall 自动 sync**，禁止人工 sync/调用 |
| `.tmp-ui-ux-pro-max/` | **不必存在** | 上游 `ui-ux-pro-max` 包的本地克隆/同步工作区 | 已在 `.gitignore`；运行时以 `.cursor/skills/ui-ux-pro-max/` 为准 |

**Skill 同步与调用（自动，禁止人工）**：

- **`npm install` / `update.js` 后的 install**：`postinstall` → `scripts/sync-skills.mjs --quiet`（Cursor 全局 + Trae `.trae/skills/`）。
- **Agent 加载**：`.cursor/rules/agent-skills-auto-exec.mdc`（`alwaysApply: true`）自动 Read 总编排与当前步骤 Skill；**勿**要求用户执行 `sync:cursor-skills` 或 `/ai-full-process-design`。
- **离线更新包**：打入 `.cursor/skills/`、`.cursor/rules/`、`scripts/sync-skills.mjs`。
- **编写规范**：`AIEP-WEB/src/skills/SKILL-STANDARD.md`（对齐 [agentskills.io](https://agentskills.io)）
- **维护 `ui-ux-pro-max`**：在仓库外 clone 上游包，将 `data/`、`scripts/` 同步到 `.cursor/skills/ui-ux-pro-max/` 后提交。

**完整性自检**（`.cursor/skills/ui-ux-pro-max/` 至少包含）：

- `SKILL.md`
- `scripts/search.py`、`scripts/core.py`、`scripts/design_system.py`
- `data/*.csv` 与 `data/stacks/*.csv`
