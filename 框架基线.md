## 架构概览

### 主壳（Shell）

- **入口**：`index.html` → `src/main.js`。
- **路由**：`src/router/index.js`，`createWebHistory`（HTML5 History）。
- **布局**：`src/App.vue` — 顶栏（首页 / 应用中心）、用户菜单（个人资料、帮助、关于、项目审查）、`<router-view>` 过渡。
- **子应用嵌入**：当路径以 `/sample-app` 开头时，主区采用嵌入布局（`isSubAppEmbed`），高度链交给内层 `SampleApp` 双栏布局。

### 主壳页面（`src/views/`）

- `Dashboard.vue`、`AppCenter.vue`、`ProfileCenter.vue`、`ReviewCenter.vue`
- `HelpPage.vue`、`AboutPage.vue`

### 示例子应用（`src/apps/sample-app/`）

- **内嵌模式**：路由挂载在 `/sample-app/...`，由 `SampleApp.vue` 提供侧栏菜单与内容区。
- **独立运行**：`src/apps/sample-app/main.js` 使用 **Hash 路由**，`build/vite.sample-app.config.js` 以 `build/sample-app.html` 为入口单独打包为 **IIFE + 单 chunk**，输出到 `dist/sample-app/`（`copy-html` 插件生成 `sample-app.html` / `index.html` / `start.html`）。
- **页面类型**：`views/` 下业务演示页；`arco-pages/` 为 Arco Pro 风格迁移页（monitor、form、profile 等）；`api/`、`hooks/`、`locales/`、`utils/chartTheme.js` 支撑子应用。

### 横切能力

- **国际化**：`src/apps/sample-app/i18n.js`，主壳与示例子应用共用。
- **主题**：`src/utils/theme.js`（`initTheme`），在 `main.js` 启动时初始化。
- **全局样式**：`src/style.css` + Arco 全量 CSS。

## 构建与脚本

| 命令 | 说明 |
|------|------|
| `npm run dev` | 主应用开发服务器（默认 Vite 端口） |
| `npm run build` | 主应用产物 → `dist/` |
| `npm run dev:sample-app` | 子应用独立开发，端口 **5174**，打开 `/build/sample-app.html` |
| `npm run build:sample-app` | 子应用产物 → `dist/sample-app/` |
| `npm run update` / `npm run rollback` | 依赖 Git 仓库与 `main` 分支；当前工作区若未初始化 git，脚本仅作参考 |

## 基线维护约定

1. **升级依赖**：修改 `package.json` 后执行 `npm install`，将变更记入 `package-lock.json`；同步更新 `framework-baseline.json` 中 `dependenciesResolved` / `devDependenciesResolved` 与 `lockedAt`。
2. **重大结构变更**：在本文档「架构概览」中补充章节，并更新 `framework-baseline.json` 的 `architecture` / `srcLayout`。
3. **发布基线**：可为 Git 打标签 `v1.1.0-baseline`（在仓库初始化后）以便与本文档交叉引用。
