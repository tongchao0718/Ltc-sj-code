# 环境安装指南

## 1. 必要环境

在开始创建和运行项目之前，需要安装以下必要环境：

| 环境 | 版本要求 | 用途 |
|------|----------|------|
| Node.js | **18.0 或更高版本**（与 Vite 6 / 本仓库一致） | 运行 Vue 项目和构建工具 |
| npm | 8.0 或更高版本 | 包管理工具 |
| Git | 任意最新版本 | 版本控制 |

## 2. 安装方法

### 2.1 Node.js 安装

#### Windows 系统

1. **下载安装包**：
   - 访问 [Node.js 官网](https://nodejs.org/zh-cn/download/)
   - 下载 Windows 安装包（建议选择 LTS 版本）

2. **运行安装程序**：
   - 双击安装包，按照提示完成安装
   - 确保勾选 "Add to PATH" 选项

3. **验证安装**：
   ```bash
   node -v
   npm -v
   ```

#### macOS 系统

1. **使用 Homebrew**：
   ```bash
   brew install node
   ```

2. **或下载安装包**：
   - 访问 [Node.js 官网](https://nodejs.org/zh-cn/download/)
   - 下载 macOS 安装包并运行

3. **验证安装**：
   ```bash
   node -v
   npm -v
   ```

#### Linux 系统

1. **使用包管理器**：
   ```bash
   # Ubuntu/Debian
   sudo apt update
   sudo apt install nodejs npm

   # CentOS/RHEL
   sudo yum install nodejs npm

   # Fedora
   sudo dnf install nodejs npm
   ```

2. **或使用 nvm**：
   ```bash
   curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
   nvm install --lts
   ```

3. **验证安装**：
   ```bash
   node -v
   npm -v
   ```

### 2.2 Git 安装

#### Windows 系统

1. **下载安装包**：
   - 访问 [Git 官网](https://git-scm.com/download/win)
   - 下载 Windows 安装包

2. **运行安装程序**：
   - 双击安装包，按照提示完成安装
   - 确保勾选 "Add Git to PATH" 选项

3. **验证安装**：
   ```bash
   git --version
   ```

#### macOS 系统

1. **使用 Homebrew**：
   ```bash
   brew install git
   ```

2. **或下载安装包**：
   - 访问 [Git 官网](https://git-scm.com/download/mac)
   - 下载 macOS 安装包并运行

3. **验证安装**：
   ```bash
   git --version
   ```

#### Linux 系统

1. **使用包管理器**：
   ```bash
   # Ubuntu/Debian
   sudo apt update
   sudo apt install git

   # CentOS/RHEL
   sudo yum install git

   # Fedora
   sudo dnf install git
   ```

2. **验证安装**：
   ```bash
   git --version
   ```

## 3. 一键安装脚本

### 3.1 Windows 一键安装脚本

创建一个名为 `install-env.ps1` 的 PowerShell 脚本：

```powershell
# 下载并安装 Node.js LTS
Write-Host "正在下载 Node.js LTS..."
Invoke-WebRequest -Uri "https://nodejs.org/dist/v20.11.1/node-v20.11.1-x64.msi" -OutFile "nodejs-installer.msi"

Write-Host "正在安装 Node.js..."
Start-Process msiexec.exe -ArgumentList "/i nodejs-installer.msi /quiet /qn /norestart" -Wait

# 下载并安装 Git
Write-Host "正在下载 Git..."
Invoke-WebRequest -Uri "https://github.com/git-for-windows/git/releases/download/v2.43.0.windows.1/Git-2.43.0-64-bit.exe" -OutFile "git-installer.exe"

Write-Host "正在安装 Git..."
Start-Process .\git-installer.exe -ArgumentList "/VERYSILENT /NORESTART /NOCANCEL /SP- /SUPPRESSMSGBOXES" -Wait

# 验证安装
Write-Host "验证 Node.js 安装..."
node -v
Write-Host "验证 npm 安装..."
npm -v
Write-Host "验证 Git 安装..."
git --version

# 清理安装文件
Remove-Item nodejs-installer.msi
Remove-Item git-installer.exe

Write-Host "环境安装完成！"
```

### 3.2 macOS 一键安装脚本

创建一个名为 `install-env.sh` 的 bash 脚本：

```bash
#!/bin/bash

# 安装 Homebrew（如果未安装）
if ! command -v brew &> /dev/null; then
    echo "正在安装 Homebrew..."
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
fi

# 安装 Node.js 和 Git
echo "正在安装 Node.js 和 Git..."
brew install node git

# 验证安装
echo "验证 Node.js 安装..."
node -v
echo "验证 npm 安装..."
npm -v
echo "验证 Git 安装..."
git --version

echo "环境安装完成！"
```

### 3.3 Linux 一键安装脚本

创建一个名为 `install-env.sh` 的 bash 脚本：

```bash
#!/bin/bash

# 检测 Node.js 是否已安装
if command -v node &> /dev/null; then
    echo "Node.js 已安装: $(node -v)"
else
    echo "Node.js 未安装，开始安装..."
    # 更新包管理器
    echo "正在更新包管理器..."
    sudo apt update || sudo yum update || sudo dnf update
    
    # 安装 Node.js
    if command -v apt &> /dev/null; then
        sudo apt install -y nodejs npm
    elif command -v yum &> /dev/null; then
        sudo yum install -y nodejs npm
    elif command -v dnf &> /dev/null; then
        sudo dnf install -y nodejs npm
    fi
fi

# 检测 Git 是否已安装
if command -v git &> /dev/null; then
    echo "Git 已安装: $(git --version)"
else
    echo "Git 未安装，开始安装..."
    # 安装 Git
    if command -v apt &> /dev/null; then
        sudo apt install -y git
    elif command -v yum &> /dev/null; then
        sudo yum install -y git
    elif command -v dnf &> /dev/null; then
        sudo dnf install -y git
    fi
fi

# 验证安装
echo "验证 Node.js 安装..."
try {
    node -v
} catch {
    echo "Node.js 安装失败"
}

echo "验证 npm 安装..."
try {
    npm -v
} catch {
    echo "npm 安装失败"
}

echo "验证 Git 安装..."
try {
    git --version
} catch {
    echo "Git 安装失败"
}

echo "环境安装完成！"
```

## 4. 使用说明

1. **下载并运行脚本**：
   - 根据你的操作系统选择对应的脚本
   - 运行脚本（Windows 系统需要以管理员身份运行 PowerShell）

2. **验证安装**：
   - 脚本运行完成后，会自动验证安装结果
   - 你也可以手动运行验证命令确认安装成功

3. **配置 npm 镜像**（可选）：
   ```bash
   # 设置 npm 镜像为淘宝镜像
   npm config set registry https://registry.npmmirror.com
   ```

## 5. 注意事项

- 确保你的系统满足最低要求
- 安装过程中可能需要管理员权限
- 如果遇到网络问题，可以尝试使用镜像源
- 安装完成后，建议重启终端以确保环境变量生效

## 6. 故障排除

### 6.1 Node.js 安装失败

- 检查网络连接
- 尝试使用不同版本的 Node.js
- 手动下载安装包并运行

### 6.2 Git 安装失败

- 检查网络连接
- 尝试使用不同版本的 Git
- 手动下载安装包并运行

### 6.3 环境变量问题

- 确保安装过程中勾选了 "Add to PATH" 选项
- 重启终端或计算机
- 手动添加环境变量

通过本指南，你可以轻松安装项目所需的 Node.js 和 Git 环境，为项目的创建和运行做好准备。
