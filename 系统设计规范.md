# 系统设计规范-Arco Design

## 一、设计理念

参考 Arco Design 的设计理念，我们将打造一个现代、简洁、易用的 UI 系统，注重用户体验和视觉美感。Arco Design 强调「简洁、一致、易用」的设计原则，通过系统化的设计变量和组件库，实现高效、美观的界面设计，详细参考示例：<https://vue-pro.arco.design/dashboard/workplace、https://arco.design/iconbox/libs、https://arco.design/docs/spec/style-guideline、https://arco.design/docs/spec/philosophy网站规则及要求。>

## 五、布局结构

### 1. 整体布局

- 顶部导航栏：固定高度 60px
- 侧边菜单：固定宽度 200px（展开）/ 64px（折叠）
- 主内容区：自适应宽度
- 页脚：固定高度 60px

### 2. 响应式布局

- 大屏幕（>1200px）：三栏布局
- 中屏幕（768px-1200px）：两栏布局
- 小屏幕（<768px）：单栏布局，侧边菜单折叠为抽屉

### 3. 栅格系统

- 12 列栅格
- 间距：16px（默认）
- 断点：
  - xs：< 576px
  - sm：576px - 767px
  - md：768px - 991px
  - lg：992px - 1199px
  - xl：≥ 1200px

## 六、动画效果

### 1. 过渡动画

- 页面切换：淡入淡出（0.3s）
- 组件出现：从下往上淡入（0.3s）
- 组件消失：从上往下淡出（0.3s）
- 模态框：从中心缩放（0.2s）
- 抽屉：从右侧滑入（0.3s）

### 2. 微交互

- 按钮悬停：轻微上浮和阴影变化（0.2s）
- 输入框聚焦：边框颜色变化和轻微阴影（0.2s）
- 菜单展开：平滑展开（0.3s）
- 卡片悬停：轻微上浮和阴影增强（0.2s）

### 3. 加载动画

- 线性加载条：蓝色渐变
- 圆形加载动画：蓝色旋转
- 骨架屏：灰色渐变闪烁

## 八、交互规范

### 1. 导航栏交互

#### 个人中心下拉菜单

- **触发方式**：点击个人中心按钮
- **关闭方式**：
  - 再次点击按钮
  - 点击菜单项后自动关闭
  - 鼠标移出菜单区域 1 秒后自动关闭（使用 setTimeout 实现）
- **防误触机制**：鼠标在按钮和菜单之间移动时，通过 mouseenter/mouseleave 事件和计时器控制，避免意外关闭

#### 主题切换

- **触发方式**：点击主题按钮
- **效果**：即时切换，平滑过渡动画 0.3s
- **存储**：自动保存到 localStorage

### 2. 弹窗交互

#### 打开/关闭

- **打开**：点击触发按钮，淡入动画 0.3s
- **关闭**：
  - 点击关闭按钮
  - 点击遮罩层
  - 按 ESC 键（可选）
- **动画**：从中心缩放或从上方滑入

#### 尺寸规范

- **宽度**：80vw（视口宽度的 80%）
- **最大宽度**：根据内容调整（HelpModal: 900px, AboutModal: 600px）
- **高度**：自适应内容，最大 80vh

### 3. 表单交互

#### 输入框

- **聚焦**：边框变为 #165DFF，添加阴影 0 0 0 2px #E8F3FF
- **错误**：边框变为 #F53F3F，显示错误提示
- **禁用**：背景色 #F2F3F5，文字色 #C9CDD4

#### 按钮

- **悬停**：轻微上浮 translateY(-2px)，阴影增强
- **点击**：轻微下沉 translateY(0)
- **禁用**：透明度 0.5，无交互效果

### 4. 卡片交互

#### 悬停效果

- **上浮**：translateY(-4px)
- **阴影**：增强为 0 4px 16px rgba(0, 0, 0, 0.12)
- **过渡**：0.3s ease

## 九、响应式设计规范

### 1. 断点定义

- **xs**：< 576px（手机）
- **sm**：576px - 767px（大手机/小平板）
- **md**：768px - 991px（平板）
- **lg**：992px - 1199px（小桌面）
- **xl**：≥ 1200px（大桌面）

### 2. 导航栏响应式

#### 大屏幕（≥768px）

- 显示完整导航链接文字
- 显示用户名称
- 应用图标 + 系统名称

#### 小屏幕（<768px）

- 导航链接只显示图标
- 用户菜单只显示图标
- 应用图标（可选隐藏名称）

### 3. 内容区响应式

#### 卡片布局

- **大屏幕**：3 列网格
- **中屏幕**：2 列网格
- **小屏幕**：1 列网格

#### 表单布局

- **大屏幕**：多列布局
- **中屏幕**：2 列布局
- **小屏幕**：单列布局

   - 完善深色模式，使用 Arco Design 的深色主题变量

   - 跨浏览器测试，确保在所有主流浏览器中正常显示
   - 响应式测试，确保在不同屏幕尺寸下都有良好的显示效果
   - 性能优化，确保页面加载和交互流畅

## 十、技术实现

### 1. CSS 变量

使用 CSS 变量管理颜色、间距等样式参数，便于主题切换和维护。参考 Arco Design 的设计令牌，定义完整的变量系统。

### 2. 响应式设计

使用媒体查询和弹性布局实现响应式设计，结合 12 列栅格系统，确保在不同屏幕尺寸下都有良好的显示效果。

### 3. 动画实现

使用 CSS 过渡和动画实现流畅的交互效果，包括页面切换、组件出现/消失、微交互等。

### 4. 主题切换

使用 JavaScript 管理主题状态，结合 CSS 变量实现主题切换，包括浅色模式和深色模式。

### 5. 组件库集成

考虑集成 Arco Design Vue 组件库，或参考其设计规范实现自定义组件，确保界面风格一致。
